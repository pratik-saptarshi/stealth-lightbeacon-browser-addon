# Stealth Lightbeacon Browser Addon v0.1.9

Release date: 2026-05-29

## Summary

`v0.1.9` is a hardening release that finalizes side-panel-first defaults, improves
browser-runtime validation reliability, and tightens CI workflow security posture.

## Highlights

- Removed `action.default_popup` from manifest to enforce side-panel-first runtime behavior.
- Updated Playwright launch strategy to prefer default Chromium runtime semantics with fallback strategy.
- Hardened strict smoke validation:
  - side-panel shell fallback when extension service workers are unavailable,
  - explicit side-panel bootstrap in fallback mode,
  - consistent filtering of known false-positive `aria-allowed-role` violations.
- Added repository hygiene ignores for transient outputs (`coverage/`, `test-results/`, `tmp/`).
- Hardened GitHub Actions workflow:
  - explicit workflow concurrency (`cancel-in-progress`),
  - per-job timeout limits,
  - `actions/checkout` with `persist-credentials: false`.

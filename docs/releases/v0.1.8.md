# Stealth Lightbeacon Browser Addon v0.1.8

Release date: 2026-05-29

## Summary

`v0.1.8` is a validation-stability release focused on making local and CI-like
test execution robust in constrained environments while preserving strict checks
where full browser execution is available.

## Highlights

- Added explicit Chromium discovery for installed-browser paths, including
  `/usr/local/bin/chromium`, across Playwright and launcher surfaces.
- Hardened smoke validation so Playwright launch failures gracefully fall back
  to jsdom + axe checks instead of hard-failing.
- Filtered the known jsdom-only false positive `aria-allowed-role` in fallback
  mode.
- Fixed popup tab-shell history-entry expectation to align with restored
  history rendering behavior.
- Added `pnpm-workspace.yaml` with `allowBuilds.esbuild=true` to support
  pnpm build-script policy workflows.

## Validation

- `PLAYWRIGHT_CHROME_EXECUTABLE_PATH=/usr/local/bin/chromium CHROME_BIN=/usr/local/bin/chromium pnpm run test:unit --coverage`
- `PLAYWRIGHT_CHROME_EXECUTABLE_PATH=/usr/local/bin/chromium CHROME_BIN=/usr/local/bin/chromium pnpm run test:integration --coverage`
- `pnpm run build`
- `PLAYWRIGHT_CHROME_EXECUTABLE_PATH=/usr/local/bin/chromium CHROME_BIN=/usr/local/bin/chromium pnpm run test:ui-load`

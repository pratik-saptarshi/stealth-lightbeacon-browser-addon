# Stealth Lightbeacon Browser Addon v0.1.10

Release date: 2026-05-30

## Summary

`v0.1.10` restores standalone scan reliability for side-panel-first runtime flows
and improves report output fidelity for executive-facing HTML/Markdown exports.

## Highlights

- Fixed side-panel scan startup when active tab URL is unavailable:
  - popup scan request now tolerates missing `activeTab.url`,
  - service worker canonicalizes request URL from extracted page context.
- Updated runtime messaging to side-panel-first language:
  - `Side panel shell loaded outside the extension runtime.`
- Aligned report rendering with fidelity-style formatting patterns:
  - richer executive summary structure in HTML exports,
  - score + verdict + domain distribution in Markdown exports,
  - improved issue table readability and diff section presentation.
- Preserved repository hygiene and CI hardening defaults.

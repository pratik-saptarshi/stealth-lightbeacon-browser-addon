# Stealth Lightbeacon Browser Addon v0.1.11

Release date: 2026-05-30

## Summary

`v0.1.11` improves scan trust by introducing a hybrid fallback policy and
strengthens standalone resilience under extension permission constraints.

## Highlights

- Added hybrid escalation policy in scan orchestration:
  - run local embedded rules first,
  - auto-escalate `dom-lite` to `stealth-playwright` when dynamic trigger conditions are met.
- Added merged-result behavior for hybrid scans:
  - preserve local deterministic findings,
  - append backend findings with dedupe for higher-trust output.
- Hardened service-worker context extraction against permission errors:
  - reinjection (`scripting.executeScript`) failures are recoverable,
  - tab messaging failures are handled gracefully where possible.
- Added regression tests for:
  - hybrid escalation + merge path,
  - permission-blocked reinjection recovery.
